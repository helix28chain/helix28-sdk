# helix28-sdk

Developer SDK for building on Helix28Chain.

## What This Will Provide

- **DID Tools** — create and manage zk-DIDs with post-quantum 
  Dilithium-3 signatures
- **ZKP Helpers** — generate and verify zero-knowledge proofs 
  for identity attributes
- **Enzyme Query Interface** — send yes/no proof queries between 
  Strand Alpha and Strand Beta
- **Guardian Tools** — create and manage Child DIDs with 2-of-3 
  multisig guardian structure
- **DA Layer Integration** — encrypt, store, and retrieve data 
  from the Celestia-compatible DA layer
- **Nullification Tools** — trigger GDPR/COPPA erasure requests

## Who This Is For

- **DeSci developers** building genomic data marketplaces and 
  research credential systems
- **Game developers** building child-safe environments with 
  cryptographic age and identity gates
- **Healthcare institutions** integrating EHR systems with 
  sovereign patient identity
- **Privacy application developers** building on a private-by-default 
  execution environment

## Language

Rust — matching the node implementation.
WASM bindings planned for Phase 1 to enable 
browser and mobile integrations.
